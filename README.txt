== Description ==

CedCommerce Integration for AliExpress is a comprehensive solution for WordPress-based online retailers seeking efficient integration with Aliexpress. From simplified product uploads to streamlined order synchronization, advanced product profiling, and real-time notifications, this plugin offers a suite of features designed to optimize and enhance the e-commerce experience. As an essential tool for global online selling, this plugin empowers retailers to expand their reach and maximize the potential of their e-commerce endeavors. 

= Main features =

* **Product Profiling And Category Mapping**
Create and manage product profiles for efficient categorization and organization on Aliexpress. Map your WooCommerce categories to corresponding Aliexpress categories for seamless product listing.

* **Option Value Mapping**
Map product attribute values and variations to ensure accurate representation on Aliexpress.

* **Products Grid**
View and manage your product catalog through an intuitive products grid within the WordPress admin. Easily upload your products on Aliexpress directly from Products grid.

* **Orders Grid**
Monitor and process your Aliexpress orders with ease using the dedicated orders grid. Efficiently synchronize your WooCommerce orders with Aliexpress, ensuring accurate tracking and management.

* **Activities Notification**
Receive real-time notifications for important activities, ensuring you stay informed about your store's operations.

* **Configuration Section**
Effortlessly configure plugin settings through the dedicated configuration section, allowing you to tailor the integration to your specific needs.

= External API =
This plugin uses external API (https://aliexpress-api-backend.cifapps.com/(endpoint)) to facilitate seamless integration with Aliexpress.The API enables advanced features such as managing product listings, order synchronization. For more information about the API, please visit Cedcommerce Privacy Policy Page (https://cedcommerce.com/privacy-policy).

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of Product Auto Share, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "CedCommerce Integration for AliExpress" and click Search Plugins. Once you've found our CedCommerce Integration for AliExpress plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".

= Manual installation =

The manual installation method involves downloading our CedCommerce Integration for AliExpress and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

== Frequently Asked Questions ==

### How do I upload products to Aliexpress?
Navigate to the 'Product Upload' section and follow the guided steps to upload your products to Aliexpress.

### How can I receive order notifications?
Ensure your notification preferences are configured in the 'Activities Notification' section to receive timely updates.
